The game is a ROM type ( .BIN ) !

PAPI MOVE :
* D-PAD 1 TO MOVE.
* START TO PAUSE THE GAME.
* BUTTON A TO FIRE.
* BUTTON B TO REZ PLAYER AND CIVILIANS
* BUTTON C TO USE KNIFE.


MAMI MOVE :
* D-PAD 2 TO MOVE.
* START TO START THE 2P MODE.
* BUTTON A TO FIRE.
