CONTROLS:
On title screen:
UP/DOWN - select option
START - accept selected option 

In game:
RIGHT - run forward
LEFT - stand
DOWN - slide on back
A, B, C - jump
A, B, C (In air) - double jump

On game over, if you have hi-score:
UP/DOWN - select letter
A, B, C - accept letter
START - end input of your name
